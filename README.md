# shepherd

### Prerequisites

 - [Node.js 4.3](https://nodejs.org/en/download/)
 - [Serviceless Framework 1.0](https://serverless.com/)
 
 

### Invoke function locally

```
serverless invoke local -f findSheep
```

### Deploy service to AWS

```
serverless deploy -v
```

### Remove service from AWS

```
serverless remove -v
```
